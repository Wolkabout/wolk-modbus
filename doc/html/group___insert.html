Detailed Description

Supported types are HOLDING_REGISTER, INPUT_REGISTER, COILS and INPUT_BITS. If you don't want to use any of registers types insert 0

◆ modbus_register_mapping_init()

void modbus_register_mapping_init ( void  )

User defined registers. Here user need to map address stack define in modbus_variables_map_t struct to Registers table.

Registers table consist following types:

  • HoldingRegister[HOLDING_REGISTER]
  • InputRegister[INPUT_REGISTER]
  • Coils[COILS]
  • InputBits[INPUT_BITS]

Example of usage

void modbus_register_mapping_init(void)
{
modbus_variable_init(&modbus_variables_map.holding_register_1, &HoldingRegister[HOLDING_REGISTER_1]);
}

modbus_variable_init() for details.
